# 🎾 ラケットスポーツ マッチメイカー

1日だけのラケットスポーツ社内イベント用マッチメイクWebアプリです。
ピックルボール・テニスなど、ダブルス前提の社内スポーツイベントで使えます。

## 機能

- イベント作成・編集（コート数、参加者、時間設定）
- 自動マッチメイク（ペア・対戦カード・タイムテーブル生成）
- スマホ対応の閲覧画面
- 自分の試合だけをフィルタ表示
- PIN認証による編集保護

## セットアップ

### 必要環境

- Node.js 18以上
- npm または yarn

### インストール

```bash
# バックエンド
cd racket-match-maker/backend
npm install

# フロントエンド
cd ../frontend
npm install
```

### 起動

ターミナルを2つ開いて、それぞれで実行してください。

**バックエンド（ターミナル1）:**
```bash
cd racket-match-maker/backend
npm run dev
```
→ http://localhost:3001 で起動

**フロントエンド（ターミナル2）:**
```bash
cd racket-match-maker/frontend
npm run dev
```
→ http://localhost:3000 で起動

ブラウザで http://localhost:3000 にアクセスしてください。

## 使い方

### 主催者

1. トップページでイベント情報を入力
   - イベント名、日付、開始・終了時刻
   - 1試合の時間、休憩時間
   - コート数
2. 参加者を追加（名前、レベル、ルール理解度）
3. マッチメイク設定を選択
4. 4桁のPINを設定
5. 「スケジュール生成」をクリック
6. 生成されたURLを参加者に共有

### 参加者

1. 共有されたURLにアクセス
2. 自分の名前を選択すると、自分の試合だけがハイライト表示
3. 各ラウンドの時間・コート・ペア・対戦相手を確認

### 編集

1. 閲覧画面で「編集モードへ」をクリック
2. PINを入力
3. 設定を変更して保存・再生成

## マッチメイクロジック

以下の優先順位で最適化を行います：

1. **ペアの重複を避ける** - 同じペアが繰り返されないように
2. **対戦相手の重複を避ける** - 同じ4人の組み合わせを避ける
3. **レベルバランス** - 近いレベル同士でペアを組む

人数がコート数×4で割り切れない場合は、余った人は休憩になります。

## 技術スタック

- フロントエンド: React + TypeScript + Vite
- バックエンド: Node.js + Express + TypeScript
- データベース: SQLite (better-sqlite3)
- スタイリング: Pure CSS (モバイルファースト)
